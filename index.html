Rapid Typer

For my first project at WDI, I created a game called, "Rapid Typer". The purpose of the game is to have players challenge one another to achieve the best score possible before the timer runs out. Each player will take turns once the game in over. The motivation to create such a game came from my 8 year old niece as she is learning how to use the keyboard for a computer apart from using a touch screen. This was a good opportunity for me to introduce a game to her in which I used to play when I was learning how to use a keyboard back in the day. I drew inspiration from a very popular online typing game called "Type Racer" as it has people all over the world competing to see who can type the fastest and the most words in a minute.


The tools that were used to create this project were HTML,CSS, and Javascript

Snack Mania

For my second project, I create a food review website for healthy snacks. This website was created to help the public know about healthier snack alternatives. This was my first full stack website where I implemented features such as login and sign up capabilities as well as having administrative rights to add,update and delete food items. In addition, public users will be able to add comments to the individual food items.

This project was created using HTML,CSS,Javascript,Node.js, express along with the usage of database(MongoDB).

Common Place

For my third project, my group and I created a concierge service website called Common Place. This project was to target condos withthe idea that its residence may not have the time to do weekly chores such as groceries, dry cleaning, pet grooming and etc. During the process of creating this project, we setup interviews to the general public on how they felt about doing weekly chores and what their expectations were if there were to be such a service like Common Places avaliable. To further demostrate what we have learned in this course thus far, we made use of the another language called ruby on rails.

The tools that were used to make this website involved using, database(postico), Bootstrap HTML, CSS, and Ruby on Rails.
